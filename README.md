# 🐍 Python 학습 기록

## 1️⃣ Python 시작과 환경 설정
- Python은 **인터프리터 언어** → 한 줄씩 바로 실행 가능
- **터미널**에서 실행:  
  ```bash
  python
````

* 버전 확인: `3.12.9`
* 종료: `exit()`
* **가상 환경 생성**:

  ```bash
  python -m venv 폴더명
  ```

  * 활성화: `venv\Scripts\activate`
  * 비활성화: `deactivate`
* **주피터 설치 및 실행**:

  ```
  bash
  pip install jupyter
  jupyter notebook
  ```
* **코랩**은 구글 드라이브 기반 → 공유 및 런타임 변경 가능

---

## 2️⃣ 파일과 노트북 활용

* Python 코드는 **파일(.py)로 관리** 가능 → 재사용, 관리 용이
* Jupyter / Colab에서는 **셀 단위 코드 실행 가능**
* 변수 단독 작성: **Jupyter/Colab에서는 마지막 값 출력 가능**, `.py` 파일에서는 `print()` 필요
* Markdown 셀 활용 → 제목, 글, 목록, 코드 작성 가능
* 깃허브에 `.ipynb` 파일 그대로 올리면 **프리뷰 가능, 실행은 안됨**
* `.gitignore`로 특정 파일/폴더는 업로드 제외 가능

---

## 3️⃣ Python 기본 타입

| 자료형                 | 예시                                | 설명                                           |
| ------------------- | --------------------------------- | -------------------------------------------- |
| **정수 (int)**        | `x = 10`                          | 소수점 없는 숫자                                    |
| **실수 (float)**      | `y = 3.14`                        | 소수점 있는 숫자                                    |
| **문자열 (str)**       | `s1 = 'Hello'`, `s2 = """여러 줄"""` | 문자/문자열, 여러 줄 가능                              |
| **불리언 (bool)**      | `flag = True`                     | 참(True)/거짓(False), `True == 1`, `False == 0` |
| **None (NoneType)** | `x = None`                        | 값이 없음, 함수 반환값 없을 때 사용                        |

### 특징

* **정수와 실수 비교**: 값 같으면 `== True`, 타입은 다름
* **문자열 표기**: `' '`, `" "`, `''' '''`, `""" """` 모두 `str`
* **여러 줄 문자열**: 긴 글, SQL, HTML 등 가독성 용도
* **불리언과 숫자**: `True = 1`, `False = 0`
* **None**: 값 없음, JS의 `null`과 유사

---

## 4️⃣ 변수 규칙

* 첫 글자: 문자(a~z, A~Z) 또는 `_`
* 숫자로 시작 불가
* 예약어 사용 불가 (`if`, `for`, `True` 등)
* 언더스코어 사용 가능, `_` / `_var` / `user_name` 등
* 권장 표기법:

  * 변수 / 함수: `snake_case`
  * 클래스: `PascalCase`
  * 상수: `UPPER_SNAKE_CASE`

---

## 5️⃣ 연산과 출력

* **출력**

  * `.py` 파일: `print(x)`
  * Jupyter/Colab: 마지막 셀 값 작성 가능
* **연산**

  * 숫자: `+ - * / // %`
  * 문자열: `+` 연결, `*` 반복
  * bool: 산술 가능 (`True + 1 = 2`)
* **비교**

  * 값 비교: `==`
  * 타입 비교: `type(x) == type(y)`
  * 객체 비교: `is`
* **JavaScript 비교와 차이**

  * JS: `==` 값 비교, `===` 값 + 타입 비교
  * Python: `==` 값 비교, `type()` + `==` 또는 `is`로 타입/객체 비교

---

## 6️⃣ 코딩 스타일

* 변수 / 함수 → `snake_case`
* 클래스 → `PascalCase`
* 상수 → `UPPER_SNAKE_CASE`
* 예약어와 비슷한 이름은 피하는 것이 가독성 좋음

---

💡 **정리**

* Python 학습 초기 단계에서는 **환경 설정 → 변수/출력 → 기본 자료형 → 연산/비교 → 코딩 스타일** 순서로 이해
* Jupyter/Colab 활용하면 **실습 + 시각적 확인** 가능 → 초반 학습에 적합
* 자료형과 변수 규칙을 지키면 **가독성 좋고 오류 없는 코드 작성 가능**
